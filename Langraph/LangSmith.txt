LangSmith is a unified observability & evalautaion platform where teams can debug, 
test, and monitor AI app performance.

What does LangSmith traces?
Input & Output
All the intermediate steps
Latency
Token usage
Cost
Error
Tags
Metadata

Core conecepts:
1) Project: AI application is called project.
2) Trace: One single execution of the whole application is called Trace.
3) Run: Execution through componenets i.e. Prompt, LLM, Parser in the example given below.
